# Les cinq favoris de Nathan - API de base pour Workfront World

Vous souhaitez en savoir plus sur ce que l’API Workfront peut réellement vous apporter ? Rejoignez Nathan Johnson, ingénieur du support technique III d’Adobe, qui décompose les principes de base et introduit les API dans le monde réel.

Au cours de cette session, Nathan explore les principes de base de l’API Workfront, ce qu’elle est, comment elle fonctionne et pourquoi elle est importante, avant d’explorer ses cinq cas d’utilisation préférés qui montrent sa puissance en action.

>[!VIDEO](https://video.tv.adobe.com/v/3492587/?learn=on&enablevpops)

Des connaissances fondamentales aux applications réelles, cette session vous donnera un chemin clair et accessible dans le monde des API Workfront, guidé par un expert qui travaille avec elles tous les jours. Que vous découvriez les API ou que vous souhaitiez développer votre boîte à outils technique, cette session vous aidera à faire le lien entre le concept et la fonctionnalité.
