# Le pouvoir de la réutilisation

Examinons comment utiliser les fragments d’expérience pour stimuler la réutilisation et l’efficacité dans votre écosystème afin d’affecter le retour sur investissement et d’accroître la vitesse. Au cours de cette session, vous apprendrez comment appliquer cette fonctionnalité afin d’améliorer la vitesse d’alimentation de l’expérience digitale de vos clients et, finalement, atteindre l’échelle pour votre entreprise.
