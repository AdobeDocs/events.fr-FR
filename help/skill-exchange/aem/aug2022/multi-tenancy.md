# Multi-tenant de session AEM Champion

La session se concentrera sur la manière dont plusieurs clients peuvent exister sur la même instance d’AEM as a Cloud Service. L’accent est mis sur l’utilisation d’Adobe Cloud Manager pour déployer plusieurs référentiels de code sur la même instance AEM. Cela inclut la configuration de différents référentiels Git ainsi que l’utilisation de sous-modules Git.
