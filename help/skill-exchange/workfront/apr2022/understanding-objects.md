# Présentation des objets dans Workfront

Comprendre les objets dans Workfront est l’un des concepts les plus importants qu’un nouvel administrateur système peut maîtriser. Une fois que vous avez compris les différents objets, et les relations entre eux, le ciel est la limite. Nous vous présenterons les nuances des objets de projet tels que les portfolios, les programmes, les projets, les tâches, les problèmes/demandes, et les objets de personne tels que les entreprises, les groupes et les équipes. Considérez cette orientation d&#39;objet 101 !
