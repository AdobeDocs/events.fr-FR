# Conseils et astuces : Champs calculés que tout le monde doit connaître et comment créer des rapports !

Avez-vous déjà fait défiler plusieurs pages d’un flux de mise à jour de projet pour trouver la date à laquelle le projet a été déplacé dans Actuel ? Il s’agit d’un excellent exemple d’informations qui peuvent être capturées dans un champ calculé qui est facilement accessible ET qui peut être utilisé dans les rapports.

Au cours de cette session, vous entendrez :

* Les calculs de champs calculés préférés d’Anthony qui faciliteront la création de rapports dans Workfront
* Comment fonctionnent les calculs pour les adapter à vos besoins
* Focus sur les champs calculés d’un formulaire personnalisé (nombre de ces exemples peuvent être utilisés pour les calculs dans des colonnes personnalisées d’un rapport. Cependant, nous nous concentrerons sur les champs calculés d’un formulaire personnalisé (nous n’aborderons pas des sujets tels que les collections ou les filtres EXIST).
