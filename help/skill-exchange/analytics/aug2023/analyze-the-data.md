# Événement d’échange de compétences août 2023 - Suivi des formations - Adobe Analytics : analyse des données

Les tableaux à structure libre sont la base flexible d’Analysis Workspace. La maîtrise de ces tableaux puissants vous permettra de progresser rapidement dans votre parcours Workspace

Les segments sont un moyen très flexible de filtrer et d’analyser en profondeur vos données. Ils peuvent être appliqués globalement, au niveau des mesures ou être utilisés comme dimensions autonomes

Les panneaux offrent des raccourcis vers des analyses complexes. En particulier, le panneau d’aperçu rapide est une excellente option pour répondre simplement aux questions auxquelles votre entreprise est confrontée
