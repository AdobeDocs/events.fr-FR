# Créateurs d’expérience en vedette

## Fonctionnalités D’Analysis Workspace Que Vous N’Utilisez Peut-Être Pas

Nous savons comment utiliser la plupart des visualisations. Nous connaissons les différents composants : prêts à l’emploi et personnalisés. Saviez-vous qu’il existe des fonctionnalités moins connues que vous pourriez utiliser pour améliorer votre rapport et votre analyse dans Workspace ?

Au cours de cette session, Kaya partagera :

* Découvrez quelques fonctionnalités de Workspace que vous n’utilisez peut-être pas dans vos projets
* Découvrez comment utiliser ces fonctionnalités pour obtenir des informations sur vous et les utilisateurs finaux de votre projet Workspace
* Obtenez quelques exemples de cas d’utilisation que vous pouvez appliquer aux projets Workspace de votre entreprise.

## Trois joyaux cachés d’Adobe Analytics à connaître

Que vous soyez un utilisateur chevronné d’Adobe Analytics ou un débutant, plusieurs fonctionnalités souvent négligées peuvent améliorer considérablement votre expérience. Gipson partagera la manière dont ces outils vous permettent de naviguer dans l’interface sans effort et de rationaliser vos activités quotidiennes.
