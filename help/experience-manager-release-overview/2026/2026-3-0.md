# Vidéo De Présentation De La Version 2026.03.0 D’AEM

>[!VIDEO](https://video.tv.adobe.com/v/3483062/?captions=fre_fr&learn=on&enablevpops)

Les fonctionnalités de cette version sont axées sur les innovations de Experience Manager Sites, Forms, Assets et Cloud Foundation et comprennent les éléments suivants :

* Sites
   * Prise en charge des mises à jour et des lancements de l’agent de production Experience

* Formulaires
   * Prise en charge des workflows pour l’éditeur IC
   * Prise en charge de l’éditeur de thèmes pour les composants principaux
   * Prise en charge des versions pour Forms

* Ressources
   * AEM Content Advisor - GA avec les applications Adobe
   * Connecteur Microsoft Office (PowerPoint, Word, Excel)

* Cloud Foundation
   * Agent de gouvernance : gestion des marques
   * Autorisations d’audit sur l’interface utilisateur tactile
   * Fonction de déconnexion OIDC
   * Suspendre les mises à jour de maintenance

* Cloud Manager
   * Serveur MCP Cloud Manager
