# Informations sur Edge Network et Mobile Personalization avec les solutions Adobe

Découvrez le fonctionnement de l’architecture d’Edge et comment déboguer les implémentations à l’aide d’Assurance
Conseils techniques sur la configuration de Mobile SDK
Comment offrir des expériences personnalisées avec Target et AJO

>[!VIDEO](https://video.tv.adobe.com/v/3433328/?learn=on)

## Points essentiels à retenir

### Focus sur Edge Network et Mobile Personalization :

* Présentation de l’architecture Adobe Edge pour la collecte et la propagation des données.
* Importance de la configuration du SDK mobile pour les expériences personnalisées.
* Utilisation de Target et de Journey Optimizer pour offrir des expériences personnalisées.

### Accent mis sur l’optimisation et Personalization

* Migration vers des versions de SDK plus récentes ou à l’aide de Mobile Edge SDK pour l’optimisation.
* Utilisation de l’extension de prise de décision double pour la personnalisation.
* Améliorez l’engagement des applications mobiles et les conversions in-app grâce à des expériences personnalisées.

### Utilisation des extensions pour des fonctionnalités améliorées

* Mise en œuvre d’extensions spécifiques telles que le consentement, l’identité et Analytics pour des fonctionnalités personnalisées.
* Configuration des extensions pour Campaign Classic et Media Messaging.
* Utilisation d’Assurance pour le débogage, la validation et la simulation des processus de collecte de données.

En se concentrant sur ces points essentiels, les entreprises peuvent améliorer leurs stratégies mobiles, l’engagement des utilisateurs et offrir des expériences personnalisées de manière efficace à l’aide des solutions Adobe.
