# Stratégie Adobe Commerce GEO : optimisation de LLM pour les expériences D2C

Rejoignez Guillermo (Memo) Raya, champion Adobe Commerce, pour une session éducative sur la manière dont la stratégie GEO pour Adobe Commerce est explorée grâce à l’optimisation de la gestion du cycle de vie des informations sur les expériences Direct to Consumer (D2C).

S’appuyant sur des implémentations réelles avec l’une des marques grand public les plus reconnues au monde, couvrant les opérations D2C sur plusieurs marchés, Memo démontre comment le contenu structuré, piloté par l’intention, au sein d’Adobe Commerce peut influencer la capacité de découverte dans les environnements pilotés par l’IA, et comment la stratégie de contenu impacte directement les performances géographiques.

>[!VIDEO](https://video.tv.adobe.com/v/3491452/?learn=on&enablevpops)

## Ce que vous apprendrez

* comprendre les principes de base de la géolocalisation : découvrez comment le contenu structuré et piloté par l’intention influence la découvrabilité pilotée par l’IA ;
* Activer les fonctionnalités prêtes à l’emploi : utilisez les fonctionnalités natives d’Adobe Commerce pour implémenter des principes de géolocalisation sans personnalisation importante.
* Aligner le contenu sur les résultats commerciaux : connecter les décisions de contenu directement aux améliorations des performances GEO
* Créer un framework de géolocalisation pratique : offrez aux parties prenantes de l’entreprise un chemin clair pour activer la stratégie de géolocalisation et mesurer l’impact du contenu

Que vous débutiez dans l’évaluation de l’optimisation de vos expériences e-commerce par le biais de fonctionnalités natives d’IA ou que vous souhaitiez optimiser vos expériences e-commerce, cette session vous offrira un cadre pratique pour activer les principes de l’optimisation de l’environnement, avec un impact commercial réel.
