# Déploiement de votre propre HTML avec des composants web

Rejoignez Raymond Camden, évangéliste développeur principal chez Adobe, pour apprendre les bases des composants web, y compris les éléments personnalisés, le Shadow DOM et les modèles HTML. Explorez des exemples concrets tels que l’incorporation de PDF et la création de tableaux triables pour améliorer vos applications à l’aide de solutions modernes et réutilisables.

## Points clés

* **Présentation des composants web** les composants web permettent aux développeurs et développeuses de créer des éléments HTML personnalisés avec leur propre apparence et interactivité, définis à l’aide de JavaScript.
* **Technologies principales** les composants web sont créés à l’aide de trois technologies principales ** d’éléments personnalisés, de shadow DOM et de modèles HTML.
* **Éléments personnalisés** Les éléments personnalisés permettent la création de nouvelles balises HTML. Ils doivent utiliser un kebab-case et inclure un tiret. Les classes JavaScript sont utilisées pour définir leur comportement.
* **DOM fantôme** le DOM fantôme fournit une encapsulation pour l’arborescence DOM d’un composant, ce qui empêche les fuites CSS et permet un style plus contrôlé.
* **Modèles HTML** les modèles HTML permettent de définir des feuilles HTML et CSS qui peuvent être clonées et ajoutées au DOM. Cependant, le présentateur préfère utiliser des emplacements plutôt que des modèles pour une meilleure distribution et une meilleure flexibilité.
* **Attributs et événements** Les éléments personnalisés peuvent avoir des attributs et des gestionnaires d’événements, tels que connectorCallback et disconnectCallback, pour gérer leur comportement lorsqu’ils sont ajoutés ou supprimés du DOM.
* **Emplacements** Les emplacements permettent l’insertion de contenu dans des composants web, prenant en charge les emplacements par défaut et nommés pour une gestion de contenu plus flexible.
* **Exemples réels** Ces exemples incluent une visionneuse incorporée PDF, des espaces réservés d’image et un trieur de tableau, présentant des applications pratiques des composants web.
* **Amélioration progressive** les composants web peuvent améliorer les fonctionnalités d’HTML existantes sans interrompre le chargement de JavaScript.
* **Étapes et ressources suivantes** la présentation suggère d’explorer la participation au formulaire, le DOM fantôme déclaratif, les attributs HTML personnalisés et le rendu côté serveur. Les ressources incluent MDN, webcomponents.org, et le livre « Web Components in Action » de Ben Farrell.
