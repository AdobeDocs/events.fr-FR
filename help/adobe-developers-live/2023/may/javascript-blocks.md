# Création pratique d’un site avec les nouveaux blocs JavaScript

Création pratique d’un site avec les nouveaux blocs JavaScript

Découvrez la composabilité de nouvelle génération, la dernière innovation d’AEM Sites. Il s’accompagne d’une toute nouvelle expérience de création et de développement pour créer les sites les plus rapides sur le web. Au cours de cette session, vous apprendrez à développer de nouveaux sites avec une composabilité de nouvelle génération à partir de zéro ; comment un projet type est configuré et structuré, comment fonctionne le workflow de développement, comment styliser le contenu, implémenter des blocs personnalisés et comment apporter vos modifications à la production en mettant l’accent sur les performances web.
