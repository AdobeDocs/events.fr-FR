# Keynote d’ouverture de Adobe Developers Live - Novembre 2023

Nouveautés et nouveautés pour les développeurs

Lors de cette session d’ouverture, enregistrée sur Adobe Developers Live : Content and Commerce le 6 novembre 2023, les experts du secteur Nishant Kapoor et Cedric Huesler dévoilent les dernières avancées d’Adobe Commerce et d’Experience Manager. Explorez l’évolution et l’avenir de ces plateformes puissantes, en mettant l’accent sur les applications du monde réel, les outils de développement à code faible et les stratégies de maintenance de sites web haute performance. Ron Nagy anime cette session enrichissante, qui offre une immersion dans les outils qui permettent d’offrir des expériences digitales attrayantes et personnalisées.
