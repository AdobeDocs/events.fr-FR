# Adobe Developers Live Headless Breakout 1 - Novembre 2023

Découvrez le potentiel des API découplées Adobe Experience Manager dans cette session de Catalina Dumitru, enregistrée en direct sur Adobe Developers Live. Découvrez les fonctionnalités étendues de ces API, notamment le balisage, le contrôle de version et la recherche de point d’entrée. Découvrez comment ils facilitent la récupération efficace du contenu pour des expériences de type application sur les canaux numériques et découvrez les applications réelles de ces API en action. Cette session est une ressource inestimable pour les développeurs qui cherchent à tirer parti de la technologie découplée pour trouver des solutions innovantes.

Poursuivez la conversation dans [Experience League Communities](https://adobe.ly/3rJfZcN).
