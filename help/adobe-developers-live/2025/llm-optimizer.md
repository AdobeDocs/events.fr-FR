# Optimisez les expériences sur site pour les agents AI avec LLM Optimizer

Découvrez comment Adobe LLM Optimizer améliore la visibilité de l’IA pour votre site web, ce qui permet de le découvrir et de le protéger contre les évolutions sans modifier le code d’origine. Découvrez comment l’optimisation à la périphérie améliore les réponses des modèles en langage volumineux, en veillant à ce que votre contenu soit entièrement visible par les agents d’IA tels que le ChatGPT. Cela permet à votre marque de capturer rapidement et en toute sécurité des opportunités basées sur l’IA, améliorant ainsi les expériences digitales sur les plateformes Adobe.

>[!VIDEO](https://video.tv.adobe.com/v/3477678/?learn=on&enablevpops)

Cette session — Optimiser les expériences sur site pour les agents d’IA avec LLM Optimizer — présente Ashutosh Shroti démontrant comment le LLM Optimizer « Optimiser sur Edge » réécrit les balises au niveau de la couche CDN pour améliorer la visibilité dans les réponses de modèles en grands langages. Découvrez comment n’importe quel site web peut être prêt pour l’agence en quelques secondes sans modifier le code d’origine, ce qui rend les sites détectables et pérennes par l’IA. Enregistré en direct depuis San José.
