# Figer, restaurer, innover : une expérience pratique avec des instantanés d’environnement de développement rapide pour le développement cloud

Découvrez comment les environnements de développement rapide (RDE) de Adobe Experience Manager Cloud Service permettent des itérations rapides et natives dans le cloud. Découvrez comment les fonctionnalités d’instantané permettent aux développeurs de figer, de restaurer et de réutiliser des états RDE complets. Cela vous permet de rationaliser les tests, de restaurer instantanément et d’accélérer les cycles de développement à l’aide du plug-in de RDE de l’interface de ligne de commande Adobe I/O. Améliorez l’efficacité de votre développement cloud et alignez-vous sur les politiques de gouvernance d’entreprise.

>[!VIDEO](https://video.tv.adobe.com/v/3477705/?learn=on&enablevpops)

Cette session — Geler, restaurer, innover : pratique avec des instantanés d’environnement de développement rapide pour le développement cloud — comprend Remo Liechti, ingénieur en développement logiciel chez Adobe, et a été pré-enregistrée pour Adobe Developers Live 2025. Remo explique comment les environnements de développement rapide (RDE) permettent une itération rapide et native dans le cloud dans Adobe Experience Manager Cloud Service et comment les nouvelles fonctionnalités d’instantané permettent aux développeurs de figer, de restaurer et de réutiliser des états RDE complets. Découvrez comment rationaliser les tests, annuler instantanément et accélérer les cycles de développement dans AEM Cloud à l’aide du plug-in de l’interface de ligne de commande Adobe I/O.
