# Explorez l’avenir de Adobe Journey Optimizer avec le canal web et l’expérimentation de contenu

Le nouveau canal web permet aux marketeurs de créer des campagnes web avec un concepteur visuel WYSIWYG. En outre, Journey Optimizer vous permet désormais de tester du contenu sur n’importe quel canal (e-mail, web, push, in-app, SMS, etc.).
