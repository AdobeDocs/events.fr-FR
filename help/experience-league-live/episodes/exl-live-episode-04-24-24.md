# Nouveau canal d’expérience basée sur du code dans Journey Optimizer

Activer les surfaces entrantes et le contrôle granulaire de la personnalisation à l’aide d’actions conçues pour les développeurs et développeuses.

>[!VIDEO](https://video.tv.adobe.com/v/3428095/?quality=12&learn=on)

**Détails** : le canal d’expérience basé sur le code dans Adobe Journey Optimizer ouvre d’énormes opportunités et de nouveaux cas d’utilisation pour les marques qui développent leurs programmes de personnalisation et d’expérimentation. Ce nouveau canal permet aux équipes d’accroître le contrôle de la personnalisation à l’aide d’actions basées sur le code HTML ou JSON, avec la possibilité de modifier des aspects granulaires des applications natives. Le canal d’expérience basé sur le code offre davantage de flexibilité et de performances pour les implémentations côté client, côté serveur, hybrides et basées sur les API.

Dans cet épisode d’Experience League Live, rejoignez les experts en produits Robert Calangiu et Brent Kostak qui discutent des nouvelles fonctionnalités et des cas d’utilisation du canal d’expérience basé sur le code. Découvrez comment les développeurs, les chefs de produit et les marketeurs peuvent composer et diffuser du contenu statique et dynamique sur le web, les appareils mobiles, les appareils connectés à la télévision, les kiosques, les distributeurs automatiques, etc.
