# Demandez aux experts : principes de base de Web SDK

Au cours de cette session, découvrez « l’avenir est maintenant » de la collecte de données web avec Web SDK. Nous vous montrerons tous les principes de base du Web SDK, notamment son raisonnement, son fonctionnement et les cas d’utilisation importants qu’il alimente. Nous recevrons les experts qui connaissent le dossier de fond en comble, de sorte que nous pourrons répondre aux questions et présenter les pratiques exemplaires.
