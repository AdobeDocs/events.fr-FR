# Tirez le meilleur parti de vos données d’événement comportemental dans Real-Time CDP et Journey Optimizer.

Découvrez comment créer des agrégats de données d’événement avec des attributs calculés pour améliorer votre segmentation, votre personnalisation et vos parcours.

>[!VIDEO](https://video.tv.adobe.com/v/3425196/?quality=12&learn=on)

Les données comportementales sont un ingrédient clé pour offrir des expériences client personnalisées, mais il peut être difficile de les exploiter pleinement. Aujourd’hui, vous pouvez vous reposer sur des systèmes, des technologues et des ingénieurs de données distincts pour créer des agrégats significatifs de données comportementales afin de fournir de superbes expériences.

Au cours de cette session Experience League Live, des experts en produits vous montreront comment les « attributs calculés » dans Real-Time CDP et Journey Optimizer peuvent vous aider à utiliser une interface utilisateur simple pour créer des agrégats comportementaux en tant qu’attributs de profil pouvant être utilisés pour une segmentation et une personnalisation améliorées.
